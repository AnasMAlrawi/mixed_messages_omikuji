# Mixed Messages O-mikuji

A script that generates a random [o-mikuji](https://en.wikipedia.org/wiki/O-mikuji).

## Technologies

* JavaScript
* Node

## Setup

To run this project dowload the script locally and run the script using node:

```
$ cd ../folder
$ node script.js
```
